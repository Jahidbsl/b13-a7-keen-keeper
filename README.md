# 🛡️ KeenKeeper – Connect, Track & Visualize

**KeenKeeper** is a modern social communication and task-tracking platform built with **Next.js**. It allows friends to stay connected while providing powerful tools like automated timelines and data visualization to keep life organized.

**Live Demo:** [https://b13-a7-keen-keeper-chi.vercel.app/](https://b13-a7-keen-keeper-chi.vercel.app/)

---

## 🌟 Overview
KeenKeeper isn't just another communication tool; it's a productivity-focused social space. Whether you're catching up with friends or analyzing your activity via the **Recharts-powered Stats**, the app ensures a seamless, high-performance experience.

## 🛠️ Tech Stack
The project is built with a focus on speed, scalability, and modern UI/UX:
- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) & [DaisyUI](https://daisyui.com/) (Component Library)
- **Icons:** [Lucide React](https://lucide.dev/) & [React Icons](https://react-icons.github.io/react-icons/)
- **Data Visualization:** [Recharts](https://recharts.org/) (PieCharts for deep insights)
- **Deployment:** [Vercel](https://vercel.com/)

## ✨ Key Features

### 📡 Smart Communication & Timeline
- **Auto Search & Sort:** The timeline page automatically sorts and filters messages or tasks, so you never lose track of important updates.
- **Dynamic Interaction:** Seamlessly communicate with friends in a responsive environment.

### 📊 Advanced Data Analytics
- **PieChart Stats:** Visit the [/stats](https://b13-a7-keen-keeper-chi.vercel.app/stats) page to see a visual breakdown of activities or interactions.
- **Visual Insights:** Beautifully rendered data using **Recharts** for better decision-making.

### ⚡ User Experience (UX)
- **Optimized Loading:** Custom `loading.js` files ensure a smooth transition between routes without flickering.
- **Graceful Error Handling:** Custom **Error Pages** to guide users back when something goes wrong.
- **Mobile First:** Fully responsive design that looks stunning on everything from a 5-inch phone to a 32-inch monitor.
