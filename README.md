# 🛡️ KeenKeeper

[![Framework](https://img.shields.io/badge/Framework-Next.js%2014-black?style=flat-square&logo=next.js)](https://nextjs.org/)
[![Styling](https://img.shields.io/badge/Styling-Tailwind%20%26%20DaisyUI-blue?style=flat-square&logo=tailwind-css)](https://tailwindcss.com/)
[![Deployment](https://img.shields.io/badge/Deployed%20on-Vercel-black?style=flat-square&logo=vercel)](https://vercel.com/)

**KeenKeeper** is a cutting-edge social communication hub designed to bridge the gap between social interaction and task-oriented productivity. Built with the **Next.js App Router**, it provides a lightning-fast interface for connecting with friends while keeping track of activities through advanced data visualization.

🔗 **Live Deployment:** [View Project](https://b13-a7-keen-keeper-chi.vercel.app/)

---

## 💎 Core Value Proposition

In an era of cluttered social tools, **KeenKeeper** provides a streamlined experience where communication meets data. It’s not just about talking; it’s about visualizing your social circle and interactions through meaningful metrics.

### 🚀 Key Capabilities

| Feature | Description |
| :--- | :--- |
| **Intelligent Timeline** | Real-time **Auto-Search** and **Dynamic Sorting** of friend lists and activities. |
| **Data Analytics** | Interactive **PieCharts** powered by Recharts for activity breakdown. |
| **Seamless UX** | Zero-flicker transitions using Next.js `loading.js` and custom `error.js` handling. |
| **Adaptive UI** | A mobile-first design philosophy using **DaisyUI** and **Tailwind CSS**. |

---

## 🛠️ Technical Architecture

KeenKeeper utilizes a modern stack to ensure scalability and maintainability:

* **Core:** `Next.js 14` (App Router) for Server-Side Rendering (SSR) and optimized routing.
* **UI Components:** `DaisyUI` built on top of `Tailwind CSS` for a consistent design language.
* **Visualization:** `Recharts` for high-performance, SVG-based data rendering.
* **Icons:** A hybrid approach using `Lucide React` and `React Icons` for a lightweight footprint.
* **State Management:** React's native Composition API and Hooks for efficient data flow.

## 🛠️ Getting Started

To set up a local development environment:

1.  **Clone & Enter:**
    ```bash
    git clone [https://github.com/Jahidbsl/b13-a7-keen-keeper.git](https://github.com/Jahidbsl/b13-a7-keen-keeper.git) && cd KeenKeeper
    ```

2.  **Install Dependencies:**
    ```bash
    npm install
    ```

3.  **Launch Development Server:**
    ```bash
    npm run dev
    ```

4.  **Production Build:**
    ```bash
    npm run build && npm start
    ```
